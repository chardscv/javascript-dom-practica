# Práctica de JavaScript - DOM

Este repositorio contiene **ejercicios prácticos de JavaScript enfocados en el DOM**.

Los ejercicios se realizan como práctica y se van subiendo
progresivamente conforme se van terminando.

## Objetivo
- Practicar JavaScript puro
- Entender y manipular el DOM
- Reforzar lógica y eventos
- Llevar un registro del progreso usando Git y GitHub

## Organización
Cada carpeta corresponde a un ejercicio distinto:

    ejercicio-1/
    ejercicio-2/
    ejercicio-3/

Cada ejercicio contiene su propio HTML, CSS y JavaScript según sea necesario.

## Nota
Este repositorio es solo para fines de **aprendizaje y práctica**.
El código puede mejorar con el tiempo conforme avance el estudio.
